title: Quick Start
description: Get up and running with Shale in minutes.
---

## 1. Install the plugin

Download and install the Shale plugin for your DCC tool:

- [Unreal Engine](/integrations/unreal-engine/)
- [Blender](/integrations/blender/)
- [SketchUp](/integrations/sketchup/)
- [Maya](/integrations/maya/)

## 2. Create an account

Sign up at [shalereview.com](https://shalereview.com) and create your organization.

## 3. Start a review session

Open your scene in your DCC tool, then launch Shale from the plugin panel. Click **New Session** to create a shareable review environment.

## 4. Invite reviewers

Share the session link directly, or generate a timed guest link for external stakeholders who don't need a full account.

## 5. Collect feedback

Reviewers annotate the 3D environment using the brush engine and comment tools. All notes are tied to scene location and camera view, and sync back to your DCC automatically.
