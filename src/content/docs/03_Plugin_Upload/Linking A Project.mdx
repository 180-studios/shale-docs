Before you can export, you need to link your Blender scene to a Shale project and asset. This is done from the **Project** tab in the Shale panel.

## Steps

1. In the **Project** tab, select your **Organisation** from the dropdown

![Blender Plugin Install 1](/images/screenshots/Shale%20Blender%20Choose%20Org%201.png)

2. Select a **Project** from the list — or click **New Project** to create one

![Blender Plugin Install 1](/images/screenshots/Shale%20Blender%20Choose%20Project%201.png)

3. Once a project is selected, choose an **Asset** — or click **New Asset**
4. If creating a new asset, enter a name in the **Shale Create Asset** dialog and click **OK**
5. Now click **Link Asset**


![Blender Plugin Install 1](/images/screenshots/Shale%20Blender%20Link%20Asset%201.png)

The panel header will update to show the linked project and asset name (e.g. `Project: Tutorial Project / Asset: SciFi Scene`).
