The Shale plugin is installed through Blender's built-in Extensions system. You'll need a Blender 4.xx version.

## Steps

Blender Plugin Link: https://plugins.shalereview.com/blender/index.json

1. Open Blender and go to **Edit → Preferences**

![Blender Plugin Install 1](/images/screenshots/Shale%20Blender%20Plugin%20Install%201.png)

2. In the left sidebar, click **Get Extensions
3. Add a remote repository using the above link,

![Blender Plugin Install 1](/images/screenshots/Shale%20Blender%20Plugin%20Install%202.png)


Once installed, Shale Plugin will appear under the **Installed** section of your Add-ons list.

After installation, the Shale panel is available in the **3D Viewport sidebar** (press `N` to open it if it's not visible) under the **Shale** tab.

## Logging In 

1. Open the **Shale panel** in the 3D Viewport sidebar (`N` key)

![Blender Plugin Install 1](/images/screenshots/Shale%20Blender%20Plugin%20Login%201.png)

2. If you're not logged in, the panel will show **Offline** with a **Login** button
3. Click **Login** — a dialog will appear asking for your **Email** and **Password**
4. Enter your Shale credentials and click **OK**
5. The panel will connect to your account and show your available organisations.

![Blender Plugin Install 1](/images/screenshots/Shale%20Blender%20Plugin%20Login%202.png)
