The Shale brush engine lets reviewers draw directly on 3D environments with expressive, layered visual notes.

## Drawing tools

| Tool | Description |
|------|-------------|
| **Brush** | Smooth, pressure-sensitive strokes for freeform markup |
| **Pencil** | Hard-edged strokes for precise linework |
| **Eraser** | Remove strokes selectively |

## Stroke properties

- **Size** — adjust brush radius
- **Opacity** — control stroke transparency
- **Hardness / Softness** — feather or sharpen stroke edges
- **Color** — integrated color picker for any markup color

## Editing strokes

Existing strokes can be **panned**, **rotated**, and **scaled** after placement for precise positioning.

## Layered annotations

Multiple reviewers can leave independent annotation layers on the same scene. Layers can be toggled on/off to review feedback individually or all at once.
