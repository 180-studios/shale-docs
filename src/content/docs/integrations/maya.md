The Shale plugin for Maya integrates the review pipeline into VFX and animation workflows.

## Requirements

- Maya 2023 or later
- Shale account with an active organization

## Installation

1. Download the Shale plugin from the [Shale dashboard](https://shalereview.com)
2. Copy the plugin files to your Maya plug-ins directory
3. Load the plugin via Windows → Settings/Preferences → Plug-in Manager

## Starting a review

Open the Shale shelf, sign in, and click **New Session** to create a review environment from your current scene.
