Shale's comment system ties written feedback directly to a position and camera angle in the scene, so reviewers can always see exactly what a note refers to.

## Creating a comment

Double click anywhere in the review environment to place a spatial note. Notes can contain:

- Written text
- Drawn annotations (via the brush engine)

You can also use the toolbar at the top of the UI, or press the "C" key to enter comment mode.

## Go-to context view

Every comment stores the **exact camera position** at the time it was created. Click **Go To** on any note to jump back to that view instantly — no hunting around the scene.

## Comment threads

Comments support threaded replies, so conversations stay attached to the relevant scene location rather than drifting into external chat tools.

## Approval states

Comments can be marked with approval states (e.g. **Open**, **In Progress**, **Resolved**, **Approved**) to track progress through a review cycle.
