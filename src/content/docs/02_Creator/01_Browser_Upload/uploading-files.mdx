You can upload many kinds of files, including: 2D Images, 3D assets, 360 degree photos, and more!

<br/>
Click on the Add Asset button in the project you'd like to add an asset to in the project page.
<br/>
![Upload An Asset](/images/screenshots/Shale%20App%20Upload%20An%20Asset%201.png)

<br/>
Now choose the type of asset you want to upload. Give it a name and drag and drop the file in, or click to browse. 
<br/>
![Upload An Asset](/images/screenshots/Shale%20App%20Upload%20An%20Asset%202.png)

<br/>
Either drag and drop, or click to browse to find a file to upload!
<br/>
![Upload An Asset](/images/screenshots/Shale%20App%20Upload%20An%20Asset%203.png)

<br/>
From here you will go on leave comments, feedback and review your asset with your team!
<br/>
